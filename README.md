 # BOM
NodeMCU v3 (ESP8266 board)
Momentary Switch * 6 (For command)
Momentary Switch * 1 (Mode Change)
9V power Jack * 1
MIDI 5Pin Jack * 1
Register 10Ω(Tx Pin)
Register 33Ω(3V Pin)
Cables 
Enclosure
